## HOW TO INSTALL (DON'T MISS A THING OR GET A ERROR):
- `git clone https://github.com/mZzgamer/GhostRecon.git` 
- `cd GhostRecon/`
- `chmod +x Grecon install-requirements.sh`
- `./install-requirements.sh`
## HOW TO RUN TOOL?
- Simply type `Grecon` on your terminal.
